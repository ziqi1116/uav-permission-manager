# SQL 脚本说明

## 执行顺序

1. **若依基础脚本**（项目要求的若依初始化 SQL，如 `ry_20250522.sql`）
2. **无人机业务一键安装**：`drone_install.sql`
3. 可选：`quartz.sql`（若依定时任务表，按需执行）

## 文件说明

| 文件 | 说明 |
|------|------|
| `ry_20250522.sql` | 若依框架基础库表、菜单、角色等，**必须先执行** |
| `drone_install.sql` | 本系统整合脚本：业务表、字典、菜单、角色、drone_user 权限、首页、模拟数据、测试用户 user1 |
| `quartz.sql` | 若依 Quartz 定时任务表，按需执行 |

## 说明

- 执行完 `drone_install.sql` 后，管理员使用 **admin** 登录，测试用户 **user1** 密码与 admin 默认一致（需 123456 请在用户管理中重置）。
- user1 拥有「无人机用户」角色，左侧可见：首页、无人机管理、飞行申请、权限查看、监控管理。
